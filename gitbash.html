GitBash

Bash Shell este cea mai utilizată interfață de linie de comandă (CLI) disponibilă. Bash este un shell Unix și un limbaj de comandă care este shell-ul implicit de conectare pentru majoritatea distribuțiilor Linux. Oferă utilizatorilor un set de instrumente pentru gestionarea sarcinilor pe un sistem Linux. Linux este un sistem de operare open-source care este utilizat pe scară largă în industria tehnologiei. Unul dintre avantajele cheie ale Linux este interfața sa de linie de comandă, care oferă utilizatorilor acces direct la sistemul de bază și le permite să execute comenzi și scripturi folosind un shell precum Bash. Comenzile Bash efectuează o gamă largă de sarcini, cum ar fi:

  • Crearea, mutarea, redenumirea și ștergerea fișierelor și directoarelor
  • Compararea și îmbinarea fișierelor
  • Editarea și manipularea textului din fișiere
  • Comprimarea și decomprimarea fișierelor
  • Programarea sarcinilor pentru a rula automat
  • Rețea și acces la distanță
